🏦 Bank Deposit Prediction

📌 Problem Statement
This project predicts whether a customer will subscribe to a bank term deposit based on their personal and financial details.

---

📊 Dataset
The dataset includes customer information such as age, job, balance, contact type, and previous campaign details.

---

🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

---

🔍 Steps Performed
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Selection  
- Model Building (Logistic Regression, Random Forest)  
- Model Evaluation  

---

 📈 Results
- Built classification models to predict deposit subscription  
- Achieved good accuracy (85%)

---

💡 Key Insights
- Age and job affects decision
